# Contributing to Erwin Hatchery

Thanks for helping with Erwin Hatchery.

This project is public but not open source. Contributions are welcome only under the terms in `LICENSE.md`.

By submitting a pull request, you confirm that:

- You wrote the contribution yourself, or have the right to submit it.
- You grant Auranord the right to use, modify, sublicense, and relicense your contribution.
- You understand that forks are only allowed for proposing changes back to this repository.
- You will not deploy, redistribute, publish, or reuse this project as a separate service or derivative project.

## Development branches

- `dev`: active development
- `testing`: test deployment
- `main`: stable release

Open pull requests into `dev` unless maintainers say otherwise.
